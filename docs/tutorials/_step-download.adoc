== Step 1 - Download text2confl

Go to the link:https://github.com/zeldigas/text2confl/releases[releases page] and download the latest archive for your platform. Extract it and define a variable pointing to the binary:

[source,shell]
----
tar xzf text2confl-*.tar.gz
T2C=$(pwd)/text2confl
----

Verify it works:

[source,shell]
----
$T2C --version
----

To use `text2confl` directly from any directory in the future, move the binary to a location on your `PATH` (e.g. `mv ./text2confl /usr/local/bin/`).
