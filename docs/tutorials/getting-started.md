# Getting started

Choose the guide that matches your Confluence deployment:

- **[First upload to Confluence Cloud](./first-upload-cloud.adoc)** - for `yoursite.atlassian.net` instances
- **[First upload to Confluence Data Center / Server](./first-upload-dc.adoc)** - for self-hosted instances

Not sure which you have? If you log in at `atlassian.net`, you're on Cloud. If you access Confluence via your company's own URL, you're on Data Center or Server.
